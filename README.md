# Number Guessing Game

## Overview

Number Guessing Game is a simple console-based game developed using JavaScript. The game generates a random number within a user-defined range, and the player attempts to guess the correct number. The application provides hints indicating whether the guess is too high or too low until the correct number is guessed or the player chooses to quit.

## Features

- User-defined range for random number generation
- Random number generation
- Interactive guessing gameplay
- Hint system for incorrect guesses
- Option to quit the game at any time
- Console-based user interaction
## Technologies Used

- JavaScript

## Workflow

1. The user enters the maximum range for the random number.
2. The system generates a random number within the specified range.
3. The user attempts to guess the number.
4. If the guess is lower than the generated number, the system prompts the user to guess a larger number.
5. If the guess is higher than the generated number, the system prompts the user to guess a smaller number.
6. The process continues until the user guesses the correct number or types `quit`.
7. Once the correct number is guessed, the game displays a congratulatory message.
